## Configuration System

- [x] Create internal/config/config.go - Config struct, LoadConfig, SaveConfig
- [x] Create internal/config/generate.go - TmuxConfig() template generator
- [x] Support ~/.tmux-helper.conf - key=value format
- [x] Configurable split sizes (via split-vertical-size, split-horizontal-size)
- [x] Configurable prefix key (via prefix setting)
- [x] Configurable mouse (via mouse setting)
- [x] Configurable terminal (via terminal setting)

## Commands Added

| Command | Description |
|---------|-------------|
| `tmux-helper config show` | Show current configuration |
| `tmux-helper config set <key> <value>` | Set a config value |
| `tmux-helper config get <key>` | Get a config value |
| `tmux-helper config edit` | Edit config in $EDITOR |
| `tmux-helper config path` | Show config file path |
| `tmux-helper apply` | Regenerate ~/.tmux.conf from config |

## Config File Format

```
# ~/.tmux-helper.conf
prefix=C-a
split-vertical-size=0.50
split-horizontal-size=0.50
mouse=true
theme=purple
terminal=screen-256color
```
