## Layout Cycling

- [x] Bind `Ctrl-a + Space` to cycle layouts (tmux built-in `next-layout`)
- [x] Implemented via tmux.conf: `bind Space next-layout`
- [x] `tmux-helper layout-next` command for CLI usage

## Implementation

Uses tmux's built-in `next-layout` command via `.tmux.conf`:
```bash
bind Space next-layout
```

Also provides CLI wrapper:
```bash
tmux-helper layout-next  # Cycles to next layout
```

Note: tmux doesn't have a native "previous layout" command, so only next is supported.
